.Topic IDH_D_NEWGEO_CLEAR
Clears all entries in the board array construction list.

.Topic IDH_D_NEWGEO_ADD_BREAK
Adds a board array row break.

.Topic IDH_D_NEWGEO_ADD_BOARD
Adds a board to the next position in the board array.

.Topic IDH_D_NEWGEO_GEOMORPHIC_LIST
List of all boards in the board array construction.

.Topic IDH_D_NEWGEO_BOARD_LIST
A list of all playing boards that qualify for the board
array position currently being filled. If this list is empty
either a row break is required or no boards qualify.

.Topic IDH_D_NEWGEO_BOARD_NAME
The name that will be used for the newly created board. You must fill
this in before the OK button will be enabled.
